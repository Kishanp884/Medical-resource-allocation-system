# EMRAS Figma Screen Designs

## Design System

### Color Palette
- **Primary**: #E53E3E (Emergency Red)
- **Secondary**: #3182CE (Medical Blue)
- **Success**: #38A169 (Green)
- **Warning**: #D69E2E (Amber)
- **Danger**: #E53E3E (Red)
- **Background**: #F7FAFC (Light Gray)
- **Text**: #2D3748 (Dark Gray)

### Typography
- **Headers**: Roboto Bold, 24px+
- **Body**: Roboto Regular, 16px
- **Labels**: Roboto Medium, 14px
- **Captions**: Roboto Regular, 12px

### Components
- **Buttons**: Rounded corners, 44px height
- **Cards**: White background, subtle shadows
- **Tables**: Clean borders, hover states
- **Alerts**: Colored banners with icons

## Design Principles

### Accessibility
- **WCAG 2.1 AA Compliance**: Color contrast, keyboard navigation
- **Screen Reader Support**: Proper ARIA labels
- **High Contrast Mode**: Emergency visibility

### User Experience
- **Progressive Disclosure**: Show details on demand
- **Consistent Navigation**: Predictable interface patterns
- **Error Prevention**: Validation and confirmation dialogs
- **Feedback Systems**: Loading states, success messages

### Performance
- **Lazy Loading**: Load data as needed
- **Optimized Images**: Compressed assets
- **Caching Strategy**: Browser caching for static resources
- **Progressive Enhancement**: Core functionality without JavaScript
